This is a Zelda-style parody game
> To play the game:
```
pip install -r requirements.txt
python main.py
```
> Control:
Menu
```
Left click to Proceed
Right click to Go back/Quit game
```
Character
```
WASD: Move
J: Attack
K: Use magic
1: Switch weapon
2: Switch magic
3: Use potion
4: Use key
Q: Quit
```
